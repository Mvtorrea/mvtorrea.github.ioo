Design

The Design stage focuses on the visual appearance of the website. This is where the website starts to look like a real product.

This is web design

A web developer (or web designer) will: Choose colors, fonts, and layout styles Create wireframes or mockups Design the user interface (buttons, menus, images) Ensure the design is user-friendly and accessible Make sure the design works on different screen sizes The goal of this stage is to create a visually appealing and easy-to-use website.
